Principais Formatações

Negrito ou destaque

Nesta frase temos um termo em destaque, usando a tag <strong>

Itálico

Nesta frase temos um termo em enfase usando a tag <em>

Texto marcado

Podemos também criar um texto marcado, usando a tag <mark> e mudar a cor usando CSS

Texto grande e pequeno

Estamos criando um texto pequeno nesse parágrafo

Texto excluído

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois.

Texto sobrescrito

Podemos usar para coisas do tipo x30+2

Texto subscrito

Podemos usar para coisas do tipo H20

Trechos de código

Essa tag pode ser usada para indicar que o texto se trata de um c;código mas não deixa o texto formatado, como por exemplo um código em python num = int(input('Digite um número')) if num %2 == 0: print(f'O número {num} é par') else: print(f'O número {num} é impar')

Pré formatando texto

Para o trecho acima sair formatado do jeito que foi escrito utilizamos <pre>, exemplo:

        
num = int(input('Digite um número'))
if num %2 == 0:
    print(f'O número {num} é par')
    else: 
        print(f'O número {num} é impar')
